- download git về trước (download thế nào thì tra chatgpt)
- install git trên máy

- mở vs code lên, mở cái folder hay dùng hoặc đang dùng mặc định lên, mở terminal của vs conde trên thanh terminal, chọn new terminal (hoặc nếu nó mở theo default rồi thì vào gõ thôi)

- git clone + <link>

- sau khi git clone xong thì compile

- dùng lệnh cd vào folder. 

- cd znt810

- cd cd trnt0810

- g++ setup1.cpp, setup2.cpp, setup3.cpp, main.cpp -o trnt

- sau khi dùng g++ compile xong rồi thì dùng lệnh .\trnt thôi, nó sẽ tự chạy file exe, còn không thì vào thằng folder click 2 lần

- hoặc nếu ngại vcđ thì copy lệnh sau dán thẳng vào terminal ngay sau khi git clone về
cd znt810; cd trnt0810; if ($?) { g++ setup1.cpp, setup2.cpp, setup3.cpp, main.cpp -o trnt } ; if ($?) { .\trnt }

- còn mà ngại hơn nữa thì dùng file trnt.exe trẫm đã compile sẵn đi cho khỏe (à mà nó đ phải malware đâu nên cứ yên tâm click 2 lần đi :D)

- còn ngại hơn hơn nữa thì xóa win bán máy tính thôi, khỏi xem nữa
